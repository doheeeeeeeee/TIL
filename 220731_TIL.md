# Web

## HTML

### 웹 사이트의 구성 요소

- 웹 사이트란
  
  - 브라우저를 통해 접속하는 웹 페이지(문서)들의 모음
  
  - 웹 페이지는 글, 그림, 동영상 등 여러가지 정보를 담고 있으며, 마우스로 클릭하면 다른 웹 페이지로 이동하는 '링크'들이 들어있다. '링크'를 통해 여러 웹 페이지를 연결한 것을 웹 사이트라고 한다.

- 웹 사이트의 구성요소
  
  - HTML => 구조
  
  - CSS => 표현
  
  - Javascript =>  동작
  
  

### 웹 사이트와 브라우저

- 웹 사이트는 브라우저를 통해 동작한다.

- 브라우저마다 약간씩 다르게 동작하는 것을 해결하기 위해 웹 표준이 등장했다.
